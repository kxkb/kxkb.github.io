---
1. Клонируем репозиторий
 
```bash
git clone https://github.com/kxkb/kxkb.github.io.git
```
 
2. Вносим правки
 
```bash
git commit -m "edit"
```
 
3. Добавляем
 
```bash
git add --all
```
 
4. Коммитим
 
```bash
git commit -m "edit"
```
 
5. Пушим
 
```bash
git push origin master
```
