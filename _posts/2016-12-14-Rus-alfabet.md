title: Git основы
---
1. Клонируем репозиторий
```shell
git clone https://github.com/kxkb/kxkb.github.io.git
```
2. Вносим правки
``` shell
git commit -m "edit"
```
3. Добавляем
```shell
git add --all
```
4. Коммитим
```shell
git commit -m "edit"
```
5. Пушим
```shell
git push origin master
```
